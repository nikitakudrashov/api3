# Конвертер валют
Программа использует api для поиска и получения информации о видеоиграх.

## Переменные окружения
Для работы скрипта нужен API-ключ от сервиса с валютами (например, [ссылка на api](https://api.rawg.io/api/games))
1. Создайте файл .env в корне проекта
2. Добавьте в .env ваш API-ключ в следущем формате:
```
API_KEY = ваш_api_ключ
```

## Как установить

Python3 должен быть уже установлен. Затем используйте pip (или pip3, есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```

## Как запустить проект
После установки, проект можно запустить комадной:
```
python main.py
```

## Параметры поиска
В коде иожно изменить параметры поиска игр.
Основные параметры:
- `genres` - жанр игр
- `tags` - дополнительыне теги
- `page_size` - количество игр
- `metacritic` - диапазон рейтинга Metacritic
